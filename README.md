# public-data

A curated collection of prompts, tools, agents, patterns, and notes — mined from open source projects and personal exploration.

## Structure

| Folder | Contents |
|--------|----------|
| `prompts/` | Reusable prompt templates and system prompts |
| `tools/` | Scripts, utilities, and automation helpers |
| `agents/` | Agent definitions and orchestration patterns |
| `patterns/` | Architectural patterns and best practices |
| `notes/` | Research notes and summaries |
| `thoughts/` | Ideas, explorations, and half-baked concepts |
| `templates/` | Project templates and scaffolding |
